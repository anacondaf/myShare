# myShare

This is source code of my first blog coding myself.
:smiley:

## Installation

:octocat:

```bash
git clone https://github.com/anacondaf/myShare-blog.git
```

OR (Recommend)

```bash
git clone git@github.com:anacondaf/myShare-blog.git
```

## Usage

- Install all dev dependencies: `npm install`

- Start: `npm start`

- Build: `npm build`
